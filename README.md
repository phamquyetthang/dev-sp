## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

Clone the repository and install the dependencies:

```bash
$ git clone https://github.com/phamquyetthang/offline-dev-tools
$ cd offline-dev-tools

$ yarn install
```

#### Development

```bash
$ yarn start
```

#### Build

```bash
$ yarn make
```
